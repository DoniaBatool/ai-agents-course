# AI Agents Development Course

Welcome to the **AI Agents Development Course** — a complete, hands-on guide to building intelligent AI agents using the **OpenAI Agents SDK**.

## What You Will Learn

- What AI Agents are and how they work
- Building your first agent with OpenAI Agents SDK
- Adding tools, memory, and multi-agent patterns
- Deploying production-ready agents on Google Cloud

## Course Structure

| Module | Topic | Level |
|--------|-------|-------|
| **Module 1** | AI Agents Fundamentals | Beginner |
| **Module 2** | OpenAI Agents SDK | Intermediate |
| **Module 3** | Advanced Patterns | Intermediate |
| **Module 4** | Real Projects & Deployment | Advanced |

## Prerequisites

- Basic Python knowledge
- OpenAI API key
- Python 3.11+ installed
